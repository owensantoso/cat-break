# Docs

Start here:

1. `CURRENT_STATE.md` - what is true today
2. `ARCHITECTURE.md` - system shape and state
3. `ROADMAP.md` - likely sequence
4. `product/specs/` - numbered `SPEC-*` source docs
5. `product/plans/` - numbered `PLAN-*` docs and `IMPL-*` briefs
6. `decisions/` - only when a choice should survive future work
7. `session-logs/` - only for meaningful work receipts
8. Generated views such as `SPECS.md` and `TODOS.md` are produced by `scripts/docs-meta update`
