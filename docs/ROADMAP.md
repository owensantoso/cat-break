# Roadmap

Use this for sequence and rationale, not as a dumping ground.

## Now

- Grill and decide MVP scope.
- Pick initial technical target: SwiftUI/AppKit native macOS app is the default recommendation.
- Prototype one transparent always-on-top overlay with a placeholder cat asset.
- Prototype active-use timer detection and a wall-clock timing toggle.
- Add visible time remaining, short test intervals, and a manual `Cat Now` trigger.
- Confirm behavior on the main display and with full-screen apps.

## Next

- Add menu bar settings for work interval, break duration, snooze duration, snooze limit, and enforcement mode.
- Add settings for timing mode and idle threshold.
- Add bundled cat media with clear license.
- Replace placeholder asset with real cat media pipeline.
- Add launch-at-login.
- Add simple persistence.
- Add a first-run explanation so the interruption feels intentional, not malware-ish.

## Later

- Activity-aware mode for specific apps/sites.
- Multi-display overlay support.
- Multiple cats or custom user-provided videos.
- Smarter break rules, schedules, focus-mode integration, and do-not-disturb windows.
- Polished onboarding, sound, animations, and accessibility settings.
